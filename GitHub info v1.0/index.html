There are a lot of people asking "What is GitHub?" online. In fact, there are over 615 million responses to that query on Google.
It's a popular name in the tech space, even if you’re not a developer (yet).

But GitHub can be confusing if you’ve never used it.
Is it a cloud storage site? A social networking site for programmers? A code-sharing site? The answer to all these questions is yes, and more.
